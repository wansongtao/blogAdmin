# 个人博客后台管理接口

> **统一说明：**

**服务器基础地址：**http://127.0.0.1:5050



## 1.登录

> **接口类型：**【POST】

> **接口地址**：

```apl
/admin/login
```

> 参数

|    字段名    | 数据类型 | 必传 |                            说明                            |
| :----------: | :------: | :--: | :--------------------------------------------------------: |
| userAccount  |  string  |  是  |       2至6位以字母开头的账号。支持字母、数字、下划线       |
| userPassword |  string  | 密码 | 6至16位以字母开头的密码。支持字母、数字、下划线、英文的.?! |

> **返回值**

```json
{
	code: 200,
	data: { // 失败data为空对象
		token
	},
	message: '登录成功',
	success: true
}
```

## 2.登出

> **接口类型：**【GET】

> **接口地址**：

```apl
/admin/logout
```

> **请求头**：【Authorization】

```
格式：'Bearer ' + token
```

> 参数

| 字段名 | 数据类型 | 必传 |              说明               |
| :----: | :------: | :--: | :-----------------------------: |
| token  |  string  |  否  | 'Bearer'+ ' (一个空格)' + token |

> **返回值**

```json
{
	code: 200,
	data: {},
	message: '退出登录成功',
	success: true
}
```
